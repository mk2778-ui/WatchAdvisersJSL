AIベースの高級時計コンシェルジュサービス

Background & Motivation

機械式時計への情熱と初心者の参入障壁の解消

新社会人や営業職など、ビジネス環境において腕時計は単なる時間を知る道具を超え、第一印象や信頼感を左右する重要なアイテムです。

しかし、機械式時計の世界は数多くのブランド、複雑な歴史や機械的特性があるため、知識が不足している初心者にとって「自分の予算やライフスタイルに合った完璧な時計」を見つけるのは非常に困難です。

これらの課題（Pain Point）を解決するため、Google Gemini API（生成AI）を導入し、まるで実際の時計専門家（コンシェルジュ）に相談しているかのように、ユーザーの条件に最適化された時計を提案するウェブアプリケーションを開発しました。

Key Features

01 AI ウォッチコンシェルジュ

Google Gemini APIを活用した対話型カスタマイズ推薦システムです。ユーザーの複雑な要件を正確に分析します。

  • 予算、好みのスタイル（ダイバーズ、ドレスなど）、好みの国の条件を入力
  • AIが最適なモデルを分析し、推薦理由を自然言語で生成
  • ユーザーに合わせた専門家レベルのフィードバックを提供

Key Features

02 公式ストア位置案内

推薦された時計を実際に購入したり試着したりできるオフラインの公式店舗を探します。

  • Google Maps API連携
  • 選択した特定の地域・都市に基づく店舗検索
  • 該当ブランドの認定された公式ディーラーおよびブティックを表示

Key Features

03 詳細検索＆フィルタリング

独自に構築されたデータベース内で、希望するカテゴリー別に時計を迅速かつ直感的に探索します。

  • 多様なブランド別の分類・探索（例：Rolex, Grand Seiko 等）
  • スタイル別の分類（ドレスウォッチ、スポーツ、クロノグラフ）

高級時計のラグジュアリーさを極大化する洗練されたUI視覚効果を適用しました。

Key Features

05 マイウォッチボックス (資産管理)

自分だけのコレクションを構成し、登録された時計の総予想資産価値をリアルタイムで管理するダッシュボードです。

  • Local Storageを通じたカスタマイズデータの保存処理
  • DBにない独自のカスタム時計の直接入力機能
  • 全コレクションの総時計価値（Total Value）のリアルタイム合算計算

Key Features

06 個人/グループ ディスカッション

AIから推薦された時計の中から「究極の1本」を選ぶ際に、思考を整理するための機能です。

  • 1人での深い思考整理からグループでの意見交換まで対応
  • 複数の候補モデルに対する比較分析と意思決定プロセスをサポート
